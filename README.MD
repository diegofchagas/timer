 ## 💻 Projeto Timer

Este projeto foi desenvolvido para usar como a técnica de Pomodoro, onde o usuario pode adicionar um nome ao seu projeto e colocar e controlar o tempo gasto na construção do mesmo.

<img src="./timer/src/assets/projeto-timer.png">

## Características

- [x] Criar um cronômetro
- [x] Temporizador de interrupção
- [x] Log do histórico do cronômetro

---


### Rodando o Projeto

### ⚙ Configurações 


1- Clone este repositorio

 $ git clone https://github.com/diegofchagas/timer

 2- Acesse a pasta do projeto no terminal/cmd

$ cd timer

3- Para instalar as dependências:
> npm i

4- Para iniciar a aplicação:
> npm run dev
